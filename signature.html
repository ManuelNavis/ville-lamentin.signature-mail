Comment l'installer dans Gmail

  1. Ouvrez Gmail → Paramètres (⚙️) → Voir tous les paramètres → onglet « Général » → section « Signature ».
  2. Cliquez « Copier la signature » ci-dessus, puis collez directement dans le champ signature de Gmail (Ctrl+V / Cmd+V).
  3. Enregistrez les modifications en bas de page. Votre nouvelle signature apparaîtra dans tous vos prochains emails !
